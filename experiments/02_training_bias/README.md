# Exploring how a change in learning rate, changes a Q Learning agent's ability

## general settings
- q learning agent
- learning rate of 0.05
- deadline of 30
- 20 quantisation levels
- regular graph
    * 30 nodes
    * 5 degrees per node
- nk landscape
    * n = 14
    * k = 3
- trained over 10,000 episodes
- tested over 1,000 episodes
